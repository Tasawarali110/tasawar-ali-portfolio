üì¶ How to Run This Portfolio

Step 1: Install Node.js

Download and install Node.js from the official website:

Download Node.js (LTS)

Choose the LTS (Long Term Support) version for Windows and follow the installation wizard.

Step 2: Verify Installation

After installing, open a new PowerShell window and run:

node --version
npm --version

You should see version numbers like v20.x.x and 10.x.x

Step 3: Install Dependencies

Navigate to this project folder in PowerShell and run:

npm install

This will download all required packages (React, Vite, TailwindCSS, etc.)

Step 4: Start Development Server

Run the following command:

npm run dev

Your portfolio will automatically open in your browser at http://localhost:3000
